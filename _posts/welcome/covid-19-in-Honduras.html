covid-19 in Honduras

This blog will introduce the COVID-19 situation in Honduras

Sen Wang true
09-03-2020

Introduction

This blog gives insights on the spread of novel Coronavirus in Honduras. It is a contagious respiratory virus, outbreak of it was first identified in Wuhan, Hubei, China in December 2019, and was recognized as pandemic by the World Health Organization (WHO) on Mar 11, 2020.

Data description

This dataset comes from R package coronavirus(Krispin and Byrnes 2020), which contains the daily summary of Coronavirus cases (confirmed, death, and recovered), by state/province. The source is from Johns Hopkins University Center for Systems Science and Engineering (JHU CCSE) Coronavirus. The period covered is from January 22 to July 31.

There are 7 vars and 150720 obs in the dataset. The var explain: date - Date in YYYY-MM-DD format province - Name of province/state country - Name of country/region lat - Latitude point long - Longitude point type - An indicator for the type of cases cases - the number of daily cases

Analysis

Figure 1: COVID-19 cases in Honduras

Figure 1 shows us the daily number of confirmed cases, recovered cases and deaths from January 22 to July 31. After an outbreak of one or two months, the epidemic is now in a stable phase.

In the Table 1, we can clearly know the average and median number of new confirmed cases, deaths and recoveries per week in Honduras.

Figure 2: The weekly cases in Honduras

Table 2 shows the number of new confirmed cases per week, the number of recovered cases, the number of deaths, the number of active cases, the mortality rate, the recovery rate and the active rate. We can very simply use the interactive function to know the maximum number of new cases in a week 6239 or the number of new deaths in a week 226 and so on. We can see that at weeks 23, 24, 26, 27, the new cases were particularly high above 5,000, but at week 28, there were only 1554 new cases, and because this data is not fully documented at week 28, we don’t have two days of data, so we can’t tell if the situation is getting better. Next part I will build a weekly cumulative table of cases in Honduras for comparison.

Figure 3: The cumlative cases in Honduras

Table 3 shows weekly cumulative cases in Honduras. In this table it is hard to know whether the epidemic is still serious, but we can see that the mortality rate is very low and the current problem in Honduras, the treatment rate is very low, which leads to a high rate of activity.

Conclusion

In general, at present the situation in Honduras tends to be stable, there is no surge, but the situation is not getting better. The low recovery rate is a big problem in Honduras.
